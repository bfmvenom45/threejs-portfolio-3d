# Three.js Portfolio 3D

A modern 3D portfolio website built with React, Three.js, and Tailwind CSS.

## Features
- 3D interactive scenes
- Responsive design
- Modern UI/UX

## Tech Stack
- React
- Three.js (@react-three/fiber, @react-three/drei)
- Tailwind CSS
- Vite

## Installation
```bash
npm install
npm run dev
```

## Deployment
```bash
npm run build
```
